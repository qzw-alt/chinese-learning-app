# 中文电影学习 App — 页面开发方案（Web MVP）

## 1. 项目信息

- **项目名称**：ChineseFlix（暂定）
- **项目类型**：Web App（H5）
- **核心功能**：用电影/视频片段学中文，逐句跟读
- **开发模式**：单 agent 执行，照此方案实施即可

---

## 2. 技术栈

| 层级 | 技术 | 说明 |
|---|---|---|
| 前端 | HTML + CSS + Vanilla JS | 最轻量，不依赖框架 |
| 视频播放器 | Plyr.js（CDN引入） | 开源、支持字幕、操作简单 |
| 录音 | 浏览器 MediaRecorder API | 原生自带，不需要库 |
| 视频托管 | YouTube Embed | 先用现有合法视频 |
| 字幕格式 | SRT | Whisper 直接导出 |
| 用户数据 | localStorage | 初期不做后端 |
| AI 评分 | 暂不实现 | 界面先做好 |

---

## 3. 页面结构

```
index.html          — 首页（电影列表）
watch.html          — 播放页（核心页面）

```

## 5. 播放页（watch.html）

### 功能（核心页面）
1. 视频播放（原声 + 字幕）
2. 字幕列表（可点击）
3. 字幕点击 → 跳转播放 + 高亮
4. 跟读按钮
5. 录音 + 播放回放
6. 收藏单词（弹出释义）
7. 字幕单句循环播放

### 布局
```
┌─────────────────────────────────────────────┐
│  ← 返回  [视频标题]                          │
├─────────────────────────────────────────────┤
│                                             │
│        ┌─────────────────────────┐         │
│        │                         │         │
│        │     视频播放器           │         │
│        │     (YouTube Embed)      │         │
│        │                         │         │
│        │  [字幕显示区]            │         │
│        └─────────────────────────┘         │
│                                             │
├─────────────────────────────────────────────┤
│  [🔊 语速: 0.75x | 1x | 1.25x]             │
├─────────────────────────────────────────────┤
│  字幕列表（可点击）                          │
│  ┌─────────────────────────────────────┐    │
│  │ 1. 你好，我想买一束花。  [▶跟读]     │    │
│  ├─────────────────────────────────────┤    │
│  │ 2. 你想要什么颜色的？  [▶跟读]        │    │
│  ├─────────────────────────────────────┤    │
│  │ 3. 我想要红色的，谢谢。  [▶跟读]      │    │
│  └─────────────────────────────────────┘    │
└─────────────────────────────────────────────┘
```

### 跟读流程（交互细节）
```
用户点击某句的"跟读"按钮
  → 弹出跟读浮层
  → 播放原声（单句循环一次）
  → 显示"开始录音"按钮
  → 用户点击 → 开始录音（3-5秒）
  → 录音结束 → 播放用户录音
  → 显示"完成"按钮
  → 用户可再次跟读或关闭
```

### 跟读浮层布局
```
┌─────────────────────────────────────┐
│  跟读练习                      [×]  │
├─────────────────────────────────────┤
│  你好，我想买一束花。                 │
│  （大号字体，高亮显示）              │
├─────────────────────────────────────┤
│  [  播放原声  ]                     │
│                                     │
│  [●  开始录音  ]  ← 录音中闪红点     │
│                                     │
│  你的录音：                          │
│  [▶ 播放回放]                        │
├─────────────────────────────────────┤
│  [  继续跟读  ]  [  完成  ]         │
└─────────────────────────────────────┘
```

### 单词点击（词汇卡片）
```
用户点击字幕中的任意单词
  → 弹出词汇浮层
  ┌─────────────────────────────────┐
  │  买                               │
  │  ────────────────────────────    │
  │  mǎi                              │
  │  动 词                            │
  │                                   │
  │  释义：to buy / purchase          │
  │                                   │
  │  [  收藏到生词本  ]               │
  │  [  关闭  ]                       │
  └─────────────────────────────────┘
```

### 实现要点
- 字幕数据格式（JSON）：
```javascript
const subtitles = [
  { id: 1, start: 0.5, end: 3.2, text: '你好，我想买一束花。', translation: 'Hello, I want to buy a bouquet of flowers.' },
  { id: 2, start: 3.5, end: 6.0, text: '你想要什么颜色的？', translation: 'What color do you want?' },
  ...
];
```
- 字幕点击：用 YouTube IFrame API 监听时间，更新当前高亮字幕
- 语速控制：YouTube IFrame API 的 `setPlaybackRate()`
- 录音：MediaRecorder API，格式用 webm
- 字幕循环：监听 `onStateChange`，到达 endTime 时 seek 回 startTime
- 数据存 localStorage（生词本 + 学习进度）

```

│   └── vocabulary.js  # 生词本逻辑
```

---

## 11. 执行顺序

1. 创建文件结构（css/ js/ 目录）
2. `style.css` — 统一样式变量和基础样式
3. `data.js` — 视频数据 + 字幕数据（第一支视频）
4. `index.html` + `app.js` — 首页
5. `watch.html` + `watch.js` — 播放页（最核心）
6. `vocabulary.html` + `vocabulary.js` — 生词本
7. 视频字幕数据录入（手填10-20句核心台词）

---

## 12. 注意事项

- **不接后端**：所有数据存 localStorage
- **不接 AI 评分**：录音回放功能先做好，评分后续加
- **YouTube Embed**：国内访问不了——等后续接入国内视频源
- **字幕时间轴**：用 YouTube 视频的字幕功能辅助获取时间点
